Etiquetas

Acá va todo el párrafo

Si quiero poner una línea entre un párrafo y otro la etiqueta es hr /

Para hacer un salto de línea (de párrafo) abrimos otro párrafo.

Si quiero ir al siguiente renglón la etiqueta es br / .
Negrita "b"

Cuando queremos hacer una cita usamos el blockquote (alínea a la derecha)

Italic es con "i" 
Preformateado es "pre" se ve como escrito a máquina

Letra chica "small"
Subrayado "u"

Para no poner "br" en cada salto de línea
se puede separar por secciones con la etiqueta "div"
Div3
Div4 en ninguna de estas puse el br /

Para poder crear un campo de texto "libre" se usa dentro de "form" la etiqueta "input"
Le podemos especificar un tipo con type dentro de la etiqueta de apertura.
"number"
"password"
"text"
"submit" agrega un botón que envía el valor del form (para cambiar el texto que muestra el botón se ingresa el value="" dentro de la etiqueta
"email"

"textarea" agrega un espacio delimitado para ingresar texto
"button" agrega un botón igual al de submit. Para cambiar el texto del botón lo ingresamos entre la apertura y cierre de etiqueta. Se le pueden pasar más elementos html que al submit
"select" nos muestra un listado de opciones, para agregar opciones lo hacemos con la etiqueta "option" dentro de la etiqueta "select". Dentro de la etiqueta de "option" tenemos que indicar el valor que contiene esa opción cuando esté seleccionada "value="""
"label" etiqueta los "inputs" y muestra en la página el string que le pasemos entre las etiquetas de apertura y cierre
"placeholder" = "hint" se ingresa como valor dentro de la etiqueta del input
"fieldset" agrupa los formularios que estén dentro de su etiqueta de apertura y cierre, delimitandolos en un espacio específico.
"legend" dentro de la etiquieta "fieldset" se puede usar legend para ponerle un título (ubicándolo entre etiqueta de apertura y de cierre)

"img" para insertar una imágen, se le pasa la fuente con la propiedad "src="" " www.fuenteDeLaImagen/com/000(ancho)/000(alto)
"a" se utiliza para ingresar url, se ingresa como atributo con "hrfef="" " la dirección a la que queremos apuntar, propia o externa y se ingresa el texto a mostrar entre la etiqueta de apertura y cierre.
"ul" y "ol" lista desordenada (con puntos) o lista ordenada (con números) respectivamente. Entre la etiqueta de apertura y cierre se se listan los elementos deseados ingresandolos dentro de la apertura y cierre de "li"
"table" se usa para crear una tabla, dentro de las etiquetas de apertura y cierre se detallan las siguientes etiquetas.
"thead" indica que es la primera fila entre las etiquetas de aper y cierre indicamos las columnas con "tr".
"tr" indica la fila, entre las etiquetas de ap y cie agregamos un "th" para ingresar la información de cada columna
"th" indica las columnas, entre las etiquetas de ap y cie ingresamos la información que queremos que muestre.
"tbody" después de cerrar "thead" indica las siguientes filas (también con "tr") y sus columnas con "td".
"td" indica las columnas, entre las etiquetas de ap y cie ingresamos la información que queremos que muestre.
"tfoot" después de cerrar "tbody" ingresamos "tfoot" que indica el cierre de la tabla. Mismo sistema que "tbody"
"tr" indica fila "th" columnas.
