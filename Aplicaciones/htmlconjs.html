DOM

Ejemplo para getElementById

Es el árbol html dentro de una página web. No es el código, sino la representación que se ve plasmada.
Contiene el html, las propiedades, el CSS y la interfaz de programación standar:
- Tiene los elementos de html definidos como objetos
- Tiene todas sus propiedades y métodos para poder accederlos
- Eventos de todos los elementos de html
El DOM es un estandar de como podemos ver, cambiar o eliminar elementos html dentro de la aplicación web

Para poder linkear el archivo html con el archivo JS, dentro de la etiqueta <script> se lo llama con src="archivo.js" después del body.

// getElementById
Dentro de la etiqueta que queramos llamar desde el archivo JS se ingresa un ID representativo.
