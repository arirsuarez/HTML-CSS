Display

Los elementos se dividen en dos formas de display, bloc (usan el 100% width) o inline (toma lo que utiliza el texto). Estás propiedades vienen intrínsecas con el elemento, pero puede ser modificado desde CSS.
Los elementos se pueden volver invisibles desde CSS con display: none;

Los bloques se muestran uno abajo del otro.
Párrafos: <p />
Títulos: <h1-6 />
Div: <div />
Formularios: <form />
Header: <header />
Footer: <footer />
Section: <section />

Los inlines se muestran uno al lado del otro.
Span: <span />
Links: <a />
Imágenes: <img />

Utilización de Pantalla

width != max-width
El atributo width indica los pixeles que ocupa el elemento.
El atributo max-width indica el porcentaje de pantalla que utiliza el elemento, aunque se indique con px.
Si utilizamos max-width y achicamos la pantalla va a seguir utilizando el mismo porcentaje que con la pantalla completa, no así el width
Podemos utilizar la posición de elemento position: fixed que mantiene el contenido fijo en la pantalla aunque que scrolleemos.

Overflow
Sirve para indicar el comportamiento de un texto en un espacio:

#over {
width: 300px
height: 300px
border: #110420;
border-style: dotted;
text-align: center;
padding: 7px;
margin : auto; (centra el elemento en el layout)
overflow: hidden (oculta el contenido que no entre en la caja);
// scroll: auto (detecta para donde necesita el scroll y lo agrega)
// -x/y (definimos scroll x o y)
}

Float
Indica donde queremos que se ubique un elemento, el resto van a ocupar la pantalla que este no ocupe.
En este caso flota a la izquierda, el resto de los elementos van a ir ubicándose a la derecha (la letra de Alright).

#float {
text-align: center;
float: left;
margin-left: 16px;
border-style: double;
padding: 7px;
border-color: #110420;
overflow: auto;
width: 300px;
height: 300px;
}
