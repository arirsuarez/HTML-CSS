Propiedades CSS

Párrafo

p {
color: #110420;
text-align: center;
text-indent: 0px; (sangría).
line-height: 1.4; (interlineado).
font-size: 1.5em; (Se suele usar em en vez de px para que vaya adaptándose proporcionalmente a la config que le dé el usuario).
font-weight: 100 al 900; (grosor de la letra, multiplos de 100 entre 100 y 900, no todos los tipos de letra aceptan los valores intermedios).
word-spacing: 3px;
text-shadow: 2px 2px 2px rgba(0, 0, 0, 0.2);
}

Links

a (Estado genérico del link) {
text-decoration: underline;
text-transform: uppercase / lowercase / capitalize;
letter-spacing: 3px;
}

a:link (Estado normal, nunca fue visitado ni se le hizo click) {
color: pink;
text-decoration: none;
border: solid 1px #d00;
}

a:visited (Estado cuando el link ya fue clickeado) {
color: blue;
border: solid 1px #d00; (Si no lo cambio acá también toma el color de borde del link, chequear si se comporta siempre así)
}

a:hover (Estado cuando posamos el mouse sobre el link) {
color: grey;
text-decoration: underline;
}

a:active (Estado del link al clickear, durante el tiempo que lo mantengamos apretado) {
color: black;
}

Listas

ul{
border: solid 1px #eee
background-color: white;
border-radius: 5px;
padding: 15px;
}
ul li (Para poder darle estilo a un elemento que esté dentro de otra etiqueta en CSS, primero se indica la clase padre y después la clase a diseñar) {
list-style-type: circle; (upper-romman=núm romanos // lower-alpha=abecedario // square)
list-style-position: inside; (determina si los marcadores, bullets, están dentro o fuera del contenido de la lista
}

Tablas

table{
width: 100%; (Porcentaje horizontal de la pantalla que ocupa)
background-color: white;
border-collapse: collapse; (Sin espacio en las líneas internas de la tabla)
}

tr:nth-child(even) (Para seleccionar filas específicas, en este caso las pares) {
background-color: #baeebd;
}

td {
border: solid 1px #ddd;
padding-left: 6px;
}

table th (primero etiqueta padre, después hija) {
text-align: start;
padding-left: 6px;
background-color: #4caf50;
color: white;
padding-top: 6px;
padding-bottom: 6px;
}
