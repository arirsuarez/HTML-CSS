Para cargar código CSS en html se ingresa dentro del head con la etiqueta "link" apuntando al archivo CSS con "href="nombreDelArchivo.CSS" y con "type="text/css".
"SELECTORES" instrucción que se utiliza en CSS para seleccionar la etiqueta de HTML, tienen el mismo nombre. Para darle estilo a una determinada etiqueta, en el archivo CSS:
body {
background-color: #código hexagesimal;
color: #códigohexa; (cambia el color de la letra)
}

Podemos customizar por id o por class ingresando como atributo dentro de la etiqueta de apertura "id="textolibre" o "class="textolibre".

Ejemplificación

En html: "id="parrafo".
En CSS: "#parrafo {
color: #ff0000;
}
Para hacer referencia al id desde CSS se lo llama con el símbolo " # "

En html: "class="spanuno".
En CSS: ".spanuno{
color: #ffffff;
}
Para hacer referencia a la class desde CSS se lo llama con el símbolo " . " .
Todos los elementos con el valor de "class="span" tendrán el mismo formato.

Cajas

Utilizamos el concepto "cajas" para el diseño (margenes, fondo, bordes, etc.)

En html: "class="caja".
En CSS: ".caja {
height: 100px; (alto y ancho de la caja).
width: 100px;
color: #eee; (cambia el color de la letra).
margin: 8px; (Afuera de la caja para con la caja superior)
padding: 8px; (Adentro de la caja para con el texto)
border-style: double;(estilos de borde para esa caja)
color: #aaf;
Para las carcterísticas de diseño siempre tiene más peso la última ingresada. Y se pueden escribir en la misma línea varias indicaciones lógicas.
Por ej: border-left: 3px solid #ffffff;
border-radious: Epx; (redondea los bordes)
}
