Grid o Grilla

CSS Grid es un sistema que se implementa en CSS3 para distribuir la información de manera más cómoda y simple.
Son celdas que nos permiten distribuir el contenido como si fuera una tabla (como en un excel).

Propiedades CSS

GRID

body {
background-color: rgb(199, 193, 193);
}

.grid {
display: grid; (Permite que todos los elementos dentro de la etiqueta grid se van a desplegar como grilla, uno abajo del otro (una sola columna con el ancho máx), también está la opción inline-grid que la columna ocupa lo que ocupa el elemento
grid-template-rows: 30px 30px 10px; (Permite indicar la altura de las filas una por una. Si no se ingresa el tamaño por default toma el alto del elemento que contiene)
grid-template-columns: 5fr 2.5fr 2.5fr; (Lo mismo se puede hacer con las columnas, se puede indicar el ancho por sobre el total del espacio con fracciones)
/* grid-template-columns/rows:
minmax(valorMin, valorMax); */ (Le da un mínimo y un máximo a las columnas filas)
repeat(cantDeVecesQueSeRepiteElParámetro, Parámetro) */ (Me permite manejar los valores agrupándolos desde el primero
/*
grid-gap: 1px 2px; (Espacio entre elementos, primero el gap de las columnas, después de las filas)
}

.element {
background-color: crimson;
}

Header

.header{
grid-column: 1 / 4;
grid-row: 1 / 3; (Las columnas y filas están divididas desde la primera (parte izq. de la pantalla/top) hasta la última (parte derecha de la pantalla/bottom) por líneas.
Podemos indicarle a los elementos como comportarse (abarcar de una línea a otra mediante estos parámetros) )
}

Grid Area

display: grid;
grid-template-areas: "header header"
"Sidebar Content"
"Footer footer";
grid-template-rows: 2fr 7fr 1fr;
gap: 1px 2px;
justify-items: stretch;
text-align: center;
}

FLEXBOX

Se utiliza también para distribuir el contenido pero a diferencia de Grid, Flexbox utiliza una sola columna o una sola fila (Grid es una grilla o tabla como un xls).
.flex {
display: flex;
flex-flow: row wrap; (indica la dirección del contenido y el comportamiento en caso de que este pase los margenes del contenedor).
justify-content: space-between; (Indica el comportamiento del contenido dentro del contenedor)
}
.pri {
order: 5;
}

.quin {
order: -1;
flex-grow: 1; Crece de acuerdo al espacio que tiene disponible para utilizar
flex-basis: 50px; Le indica un tamaño inicial
flex: 1 50px; (Primero va el flex-grow y después flex-basis)
align-self: center; (Alinea el contenido con respecto a su eje principal)
}
